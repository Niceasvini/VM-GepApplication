Prompt para Criar Site Profissional de Análise Automática de Currículos com IA
Objetivo do Site
Criar um site moderno e responsivo para otimizar o processo de recrutamento por meio da análise automática de currículos com inteligência artificial, com funcionalidades de upload, ranking, análise detalhada, e painel administrativo.

Nome do Site e Identidade Visual
Nome: Viana e Moura - Recrutamento Inteligente

Exibir a logo oficial da empresa "Viana e Moura" no cabeçalho e favicon.

Design limpo, profissional, com cores sóbrias e fontes elegantes (ex: azul escuro, cinza, branco).

Layout responsivo, compatível com desktops, tablets e smartphones.

Funcionalidades Principais
1. Autenticação e Gestão de Usuários
Cadastro/login de recrutadores e administradores via Supabase Auth (e-mail/senha e OAuth opcional).

Permissões: recrutadores comuns (acesso às vagas e análises) e admins (gestão de vagas e usuários).

2. Gerenciamento de Vagas
Interface para criar, editar e excluir vagas, com campos: nome da vaga, descrição, requisitos, DCF (Documento de Conteúdo Funcional).

Visualização de todas as vagas criadas em painel com filtros e busca.

3. Upload e Armazenamento de Currículos
Upload de múltiplos currículos por vaga, formatos suportados: PDF, DOCX, TXT.

Armazenar os arquivos no Supabase Storage e metadados no banco (nome, vaga associada, status da análise, data do upload).

4. Análise Automática com IA
Integração com backend Python (ou serverless) que recebe os currículos, executa análise via modelo de linguagem, e gera score de compatibilidade de 0 a 10 (múltiplos de 0.5).

Salvar o score e o texto gerado pela IA para cada currículo.

5. Ranking e Visualização dos Candidatos
Exibir ranking dos candidatos por vaga, ordenado pelo score IA, com filtro por score mínimo.

Lista resumida dos candidatos com nome, score, e status.

Ao selecionar um candidato:

Mostrar tabela com informações extraídas do currículo (dados básicos, habilidades).

Mostrar resumo gerado pela IA (texto explicativo).

Mostrar análise detalhada (feedback do modelo, pontos fortes, pontos fracos).

Opção para marcar candidato como “Interessado”, “Reprovado” ou “Entrevistar”.

6. Dashboard e Relatórios
Painel geral com estatísticas: número de vagas, candidatos por vaga, média de scores, status dos candidatos.

Gráficos e indicadores visuais (barras, pizza, linhas) para ajudar no acompanhamento do recrutamento.

Exportar relatórios em CSV/PDF com os resultados.

Tecnologias e Arquitetura Sugeridas
Frontend: React (Next.js ou Vite) + Tailwind CSS (design moderno e responsivo).

Backend: API serverless (Node.js, Python FastAPI, ou Supabase Edge Functions) para orquestração da análise IA.

Banco de Dados: Supabase (Postgres) para dados de usuários, vagas, currículos e resultados.

Storage: Supabase Storage para arquivos de currículo.

Autenticação: Supabase Auth para login e permissões.

IA: Integração com modelo de linguagem (OpenAI GPT ou outro) para análise dos currículos.

Hospedagem: Vercel, Netlify ou similar para frontend; Supabase para backend + storage.

Visual e UX/UI
Cabeçalho fixo com logo “Viana e Moura” à esquerda e menu simples à direita (Vagas, Ranking, Dashboard, Perfil).

Tela inicial com resumo do sistema e botão para criar vaga nova.

Tela de vaga com:

Upload de currículos por drag & drop.

Lista rápida dos candidatos e seus scores (com filtros).

Seletor para exibir detalhamento do candidato.

Visualização detalhada do candidato em painel lateral ou modal elegante.

Dashboard com gráficos claros e interativos.

Cores: azul escuro, branco, cinza claro, detalhes em amarelo/dourado para destaques.

Tipografia: fontes modernas e legíveis (ex: Inter, Poppins, Roboto).

Funcionalidades Extras Sugeridas
Notificações por e-mail quando nova análise estiver pronta.

Histórico de alterações para vagas e candidatos.

Sistema de comentários entre recrutadores sobre candidatos.

Feedback dos candidatos (upload de respostas ou questionários).

Multilinguagem (português e inglês).

Exportação e integração com sistemas de RH via API.
